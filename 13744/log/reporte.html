Reporte Final del Proyecto

Conclusiones

Durante este trabajo pudimos aplicar conocimientos teóricos y prácticos sobre la computación en la nube. Logramos configurar correctamente una máquina virtual en Azure, instalar servicios y desplegar contenido web. El trabajo en equipo y la planificación fueron claves para completar las tareas sin errores significativos.

Resumen de Arquitectura

Utilizamos una arquitectura simple basada en una VM Linux en Azure, configurada como servidor web mediante Apache. El contenido se desplegó desde un repositorio remoto (Github). Esta arquitectura es escalable y fácil de mantener.

Seguridad Aplicada

Durante la implementación, aplicamos varias prácticas de seguridad, tales como:

  • Acceso restringido mediante el uso de Bastión de Azure, evitando puertos RDP o SSH abiertos públicamente.
  • Uso de grupos de seguridad de red (NSG) para controlar el tráfico entrante y saliente a la máquina virtual.
  • Conexión segura mediante HTTPS y protocolos cifrados al momento de trabajar con Git y la terminal.
  • Actualización de paquetes del sistema operativo antes de iniciar tareas de instalación (en la maquina Linux).

Reflexión Final

Este proyecto nos permitió comprender el ciclo completo del despliegue de una aplicación web en la nube, desde la creación de infraestructura hasta su visualización. A través de herramientas como Azure, Git y Apache, reforzamos habilidades clave de la carrera ya que era contenido pasado en ramos anteriores, incluyendo automatización, seguridad, administración de servidores y trabajo en equipo.
